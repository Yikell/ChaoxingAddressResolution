# ChaoxingAddressResolution
该网盘最近在网上特别火，因为它的使用体验，真的超级好。很多人看到这个名字，可能会比较眼熟。它拥有100GB免费存储空间，上传下载也同样是不限速的。
因为是加密链，您在游览器获取到的下载地址是不能直接分享或下载的，所以特意做了个解析网站项目。
为了保证能够长期使用，所以暂不公开核心的算法，只公开基本的算法。
